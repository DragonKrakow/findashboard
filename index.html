🔄 Refresh AI Data

The AI refresh triggers a GitHub Actions workflow that:

  • Fetches the latest financial news from RSS feeds (Reuters, Bloomberg, CoinDesk…)
  • Classifies and summarises each article using Groq LLaMA-3 AI
  • Updates data/news.json, data/signals.json, and data/market.json
  • Commits and pushes — GitHub Pages re-serves fresh data within seconds

Click the button below to open the workflow page on GitHub, then click Run workflow. You must be signed in with write access to the repository.

Why not directly from the browser?
Calling AI APIs directly from client-side JavaScript exposes your secret API key to every visitor. This dashboard uses a static-first architecture: the workflow runs server-side in GitHub Actions with secrets stored securely, and only the pre-generated JSON files are served to the browser. The original reference HTML (FinDashboard_Pro.html) demonstrates the browser-side pattern, but it requires a key embedded in the page and is not suitable for production.
